Questions à Choix Multiples - Réponses

Question Réponse Justification
1) Le HTML est : c) Un langage de balisage ; Le HTML est un langage de balisage utilisé pour structurer le contenu des pages web en utilisant des balises.
2) Quelle est la balise qui permet d’insérer un lien ? a) <a> La balise <a> est utilisée pour insérer un lien hypertexte dans une page HTML.
3) Quelle est la balise qui permet de créer un entête dans un tableau ? c) <th> </th> La balise <th> est utilisée pour créer un entête de colonne ou de ligne dans un tableau HTML.
4) Quelle est la méthode représente la notation en hexadécimal ? a) #FCA50B La notation en hexadécimal commence par le caractère # suivi de six chiffres ou lettres représentant les valeurs RGB.
5) Quel est le code html qui permet de créer une ancre située dans une autre page ? b) <a href="IAI.html#licence 1"> Licence1 </a> Pour créer une ancre pointant vers une section spécifique d'une autre page, il faut spécifier le nom de l'ancre après le nom du fichier HTML dans l'attribut href de la balise <a>.
6) Lequel de ces formats n’est pas un format d’image ? d) MP3 MP3 est un format de compression audio, pas un format d'image. Les autres options sont tous des formats d'image courants.
7) Quelle est la balise qui permet de regrouper les champs d’un formulaire ? c) <fieldset> </fieldset> La balise <fieldset> est utilisée pour regrouper les champs d'un formulaire HTML.
8) L’attribut colspan permet de fusionner a) Les colonnes L'attribut colspan permet de fusionner les cellules sur plusieurs colonnes dans un tableau HTML.
9) Le code suivant permet de changer la bordure et la couleur d’un tableau : c) <table border="5" bordercolor="red"> Pour définir la bordure et la couleur d'un tableau, on utilise les attributs border et bordercolor dans la balise <table>.
10) Quelle est le code qui permet de rendre un champ obligatoire ? a) Input type = "required " Pour rendre un champ obligatoire dans un formulaire HTML, on utilise l'attribut "required" dans la balise <input>.
11) Quelle est la balise qui permet de regrouper les éléments par centre d’intérêt ? b) <select> </select> La balise <select> est utilisée pour créer une liste déroulante où l'utilisateur peut choisir une option parmi plusieurs.
12) Quel est le code qui permet de créer le bouton soumettre ? a) <input type="submit" value="envoyer"/> La balise <input> avec l'attribut type="submit" crée un bouton de soumission dans un formulaire HTML.
13) Quelle est la propriété qui permet d’appliquer une image de fond ? a) Background-image La propriété "background-image" en CSS est utilisée pour définir une image de fond pour un élément HTML.
14) Comment insérer le commentaire en CSS ? b) /*ceci est un commentaire */ En CSS, les commentaires sont insérés entre /* et */.
15) Les marges intérieures utilisent la propriété suivante : a) padding La propriété "padding" en CSS est utilisée pour définir les marges intérieures d'un élément.
16) Le texte peut dépasser le bloc d’un paragraphe, pour remédier à cela on utilise la propriété : b) overflow La propriété "overflow" en CSS permet de spécifier comment gérer le contenu qui dépasse les dimensions d'un élément.
17) Les balises de types blocks possèdent une hauteur et une largeur, les propriétés permettant de les dimensionner sont les suivantes : a) width, d) height Les propriétés "width" et "height" en CSS sont utilisées pour définir respectivement la largeur et la hauteur des éléments de type block.
18) Quelle est la ligne de code permettant de joindre le fichier HTML au fichier.css ? a) <link rel="stylesheet" href="fichier.css" /> La balise <link> avec l'attribut rel="stylesheet" est utilisée pour lier un fichier CSS à un fichier HTML.
19) Quelles sont les différentes valeurs que peut prendre la propriété « text-decoration » ?
  • souligné (underline)
  • barré (line-through)
  • surligné (overline)
 La propriété "text-decoration" en CSS peut prendre différentes valeurs pour modifier l'apparence du texte, telles que souligné, barré et surligné.
20) Quelles sont les différentes options pouvant être appliquées à une image de fond ?
  • background-image
  • background-repeat
  • background-position
  • background-size
  • background-attachment
 Différentes options peuvent être appliquées à une image de fond en CSS pour contrôler sa répétition, sa position, sa taille et son comportement de défilement.
